# SIB protocol
Code snippets by me for statistical consulting project for V Meir. 
Simultaneously Integrated Boost (SIB) is an advanced radiation therapy technique used in dogs with sinonasal cancer to deliver a higher dose of radiation to the most active part of the tumor (Gross Tumor Volume, or GTV) while delivering a lower dose to surrounding tissues (Clinical Target Volume, or CTV) in the same session.
These codes are specific to the project done and may not be applicable generally. Attribution and permission required if used.
